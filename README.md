# Chatroom by nodejs

## Target

Users can start to talk in a chat room only if you type your name and the chat room's name.

## Tools

### frontend
1. vue, vue-router
2. socket.io-client

### backend
1. nodejs, express
2. socket.io

## Run on local

1.  backend
    ```sh
    cd server && npm ci && npm start
    ```
2. frontend
    ```sh
    cd client && npm ci && npm run dev
    ```
